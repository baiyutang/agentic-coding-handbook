# Choosing Your Stack

The [methodology](/guides/) matters more than the tool — but the tool and model you pick
set your ceiling. This section helps you choose deliberately instead of defaulting to
whatever you installed first.

## In This Section

- **[AI Coding Tools Compared](/stack/tools)** — CLI agents, AI-native IDEs, and editor
  extensions: how they differ and how to choose.
- **[Choosing a Model](/stack/models)** — Match the model to the task. Strong models for
  reasoning, fast models for volume.
- **[Cost & Efficiency](/stack/cost)** — Understand what you're paying for and how to get
  more out of every token.
- **[The 10x Engineer](/stack/going-10x)** — A checklist that turns these choices and the
  methodology into a real productivity multiplier.

## A Note on a Fast-Moving Space

AI coding tools and models change monthly. This section deliberately focuses on **durable
trade-offs and how to reason about them** rather than a feature-by-feature scorecard that
would be stale in a quarter. Always confirm specific capabilities, limits, and pricing
against each tool's current docs before committing.
