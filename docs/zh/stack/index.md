# 选型

[方法论](/zh/guides/)比工具更重要 —— 但你选的工具和模型，决定了你的上限。这一章帮你**有意识地选择**，而不是随手装了哪个就用哪个。

## 本章内容

- **[AI 编码工具对比](/zh/stack/tools)** —— CLI agent、AI 原生 IDE、编辑器插件：区别在哪、怎么选。
- **[模型选择](/zh/stack/models)** —— 让模型匹配任务。推理用强模型，走量用快模型。
- **[成本与效率](/zh/stack/cost)** —— 搞清你到底在为什么付费，以及如何榨干每一个 token 的价值。
- **[迈向 10x 工程师](/zh/stack/going-10x)** —— 一份清单，把这些选择和方法论变成真正的生产力倍增。

## 关于一个快速变化的领域

AI 编码工具和模型每个月都在变。本章刻意聚焦于**经得起时间的权衡，以及如何思考这些权衡**，而不是一张过一个季度就作废的功能对照表。在真正投入之前，请以每个工具的**最新官方文档**为准，核对具体能力、限制和价格。
